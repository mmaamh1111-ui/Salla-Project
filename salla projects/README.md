# 📊 Sales Performance Dashboard | Power BI Project

## 👨‍💻 About This Project
This project is a full business-focused data analysis dashboard built using Power BI.  
The goal was not only to visualize data, but to transform raw sales data into **actionable business insights** that can support decision-making, improve revenue, and identify performance gaps.

---

## 🎯 Business Problem
Companies often have large amounts of sales data but struggle to:
- Identify what drives revenue growth
- Understand customer behavior
- Detect underperforming products or categories
- Make data-driven decisions instead of assumptions

This project was designed to solve these problems.

---

## 🛠 Tools & Technologies
- Power BI (Data Visualization & Dashboarding)
- Excel (Data Cleaning & Preparation)
- SQL (Data Extraction & Querying)
- Python (Optional Data Analysis / Preprocessing)

---

## 🔄 Workflow (What I Did Step-by-Step)

1. Collected and structured raw sales data  
2. Cleaned and prepared data (removed duplicates, handled missing values)  
3. Built data model and relationships in Power BI  
4. Created calculated measures (KPIs using DAX)  
5. Designed interactive dashboard for business users  
6. Extracted insights to support decision-making  

---

## 📊 Key KPIs Analyzed
- Total Revenue
- Total Sales Volume
- Top Performing Products
- Worst Performing Products
- Sales Trends Over Time
- Category Performance
- Customer Purchasing Behavior

---

## 📈 Key Insights (MOST IMPORTANT SECTION)

- 📌 A small number of products contributed to the majority of total revenue (Pareto effect 80/20 rule)  
- 📌 Certain categories showed consistent decline in sales over time, indicating potential demand issues  
- 📌 Seasonal trends were identified, showing peak sales periods that can be used for marketing campaigns  
- 📌 Some high-volume products had low profit contribution, highlighting pricing or cost inefficiencies  
- 📌 Customer behavior analysis showed repeat purchase patterns that can be used for retention strategies  

---

## 💡 Business Impact

This dashboard can help a business to:

- Increase revenue by focusing on high-performing products  
- Reduce losses by identifying weak or low-profit items  
- Improve marketing strategy based on seasonal demand  
- Enhance customer retention using behavior insights  
- Make faster and data-driven decisions instead of guessing
